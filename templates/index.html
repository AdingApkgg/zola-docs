您的一站式静态站点引擎

告别依赖。所有功能尽在一个二进制文件中。

开始使用

无依赖

Zola 是一个单一的可执行文件，内置了 Sass 编译、语法高亮、目录生成 以及许多传统上需要设置开发环境或添加 JavaScript 库才能实现的功能。

极速

平均每个站点的生成时间不到一秒， 包括 Sass 编译和语法高亮。

可扩展

Zola 将您的整个站点渲染为静态文件，这使得处理任何流量都变得轻而易举， 而且无需担心管理服务器或数据库。

易于使用

从 CLI 到模板引擎，一切都设计得直观易用。 别只听我说，看看 文档 自己体验一下。

灵活

Zola 不会妨碍您，让您可以专注于内容，无论是博客、 知识库、落地页还是它们的组合。

增强的 Markdown

Zola 自带 shortcodes 和 内部链接 功能，让您更轻松地编写内容。
