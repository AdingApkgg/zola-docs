Cela

Cela 是一个简单、轻量级的 Zola 主题，灵感来自 Hugo PaperMod。

样式表改编自 Catppuccin。 如果你喜欢它，请在 GitHub 上给它一个 🌟。谢谢！

主题特性

  • Catppuccin 配色主题
  • 亮色/暗色模式切换
  • MathJax 支持
  • 博客 RSS 订阅
  • 全文搜索
  • 机器人工具
  • 博客归档（按年份分组）
  • 国际化 (i18n)

标签、分类和分类法

Cela 提供了类似 Hexo/Hugo 的 tags（标签）和 categories（分类），与 Zola taxonomies（分类法）兼容。在 Front Matter 中：

[taxonomies]
tags = ["Rust", "Zola"]
categories = ["Programming"]

或者 YAML 风格：

taxonomies:
  tags: ["Rust", "Zola"]
  categories: ["Programming"]

推荐使用 Zola taxonomies，因为它在构建内容结构方面更强大。更多信息请参阅 Zola 分类法。

快速开始

如果你只需要安装主题，请跳到主题安装部分。

Zola 安装

# macOS
brew install zola
# Alpine Linux
apk add zola
# Arch Linux
pacman -S zola
# Docker
docker pull ghcr.io/getzola/zola:v0.19.1

创建 Zola 站点

创建你的第一个 Zola 站点。

如果 myblog 已经存在但只包含隐藏文件（如 .git），Zola 也会填充该站点。

zola init myblog
# 或者
# 填充当前目录
zola init

你在初始化期间所做的任何选择稍后都可以在 config.toml 文件中更改。

主题安装

通过 Git 子模块

git submodule add https://github.com/edwardzcn-decade/cela themes/cela
git submodule update --init --force --recursive
git submodule sync

然后在你的 config.toml 文件中设置 theme。

theme = "cela"

通过下载发布版

  1. 从 Cela 发布页面下载最新的发布归档文件。
  2. 解压到 Zola 项目中的 themes/cela。
  3. 在 config.toml 中设置 theme。
  4. （可选）如果你是重新开始，请删除 content/ 下未使用的示例内容。
