Zola Futu

Zola-futu 是一个用于 Zola SSG 的干净快速的主题。它是 Futu 的移植版。

在此处查看在线预览：https://inhzus.io.

快速开始

# 1. 克隆仓库
git clone https://github.com/inhzus/zola-futu

# 2. 进入克隆目录
cd zola-futu

# 3. 本地服务站点
zola serve

# 4. 在浏览器中打开 http://127.0.0.1:1111/

有关更多详细说明，请访问关于安装和使用主题的 文档 页面，并访问此 config.yaml 作为示例完整版配置。
