Boring

Zola 的极简主题，由 TailwindCSS 驱动。

演示

https://boring-zola.netlify.app/

设置

在你的 zola 站点目录中：

  • 获取主题

    git submodule add https://github.com/ssiyad/boring themes/boring

  • 构建 CSS

    cd themes/boring
yarn install --frozen-lockfile
yarn build

  • 更改特定于主题的变量。它们列在 config.toml 的 extra 部分中。

有关进一步说明，请参阅 Zola 文档。
