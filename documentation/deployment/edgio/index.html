🔗手动部署

对于命令行手动部署，请按照以下步骤操作：

  1. 安装 Edgio CLI:
npm i -g @edgio/cli
  1. 在你的项目根目录创建一个包含以下内容的 package.json：
npm init -y
  1. 使用以下命令初始化你的项目：
edgio init
  1. 将项目根目录下的 routes.js 更新为以下内容：
// This file was added by edgio init.
// You should commit this file to source control.

import { Router } from '@edgio/core/router'

export default new Router().static('public')
  1. 构建你的 zola 应用：
zola build
  1. 部署！
edgio deploy
